负数的二进制

示例：-5
1、首先得到5的二进制数101,然后补零得到(一个整数按照绝对值大小转换成的二进制数，
是为原码):
  原码：0000 0101
2、取反(0变1，1变0)，得到反码:
  原码：0000 0101
  反码：1111 1010
3、然后加1(反码加1叫补码):
  原码：0000 0101
  反码：1111 1010
  补码: 1111 1011
补码(1111 1011)就是-5在计算机中的二进制表示方法，上面得到的是8位的-5，
那么如果要表示16位的-5，在左边添上8个1即可:
  1111 1111 1111 1011

现在如果知道了一个二进制数，求它的十进制10数(对负数)
  一个负数的二进制数: 1111 0011
  按着上面的反方向来，先减1: 
  负数: 1111 0011
  减1 ：1111 0010
  取反:
  负数: 1111 0011
  减1 ：1111 0010
  取反: 0000 1101

  取反得到的数(0000 1101)计算出来得到13，那么这个负数就是-13

小数与二进制转换:

示例6.25
6的二进制110，小数位换算(每次乘2, 不足1为0)
  0.25*2 = 0.5 ==> 0
  0.5*2  = 1.0 ==> 1
最后得到6.25的二进制数: 110.01

二进制转小数:
 0.01 ==> 2^0 * 0 + 2^-1 * 0 + 2^-2 * 1 = 0.250

运算符

&(按位与运算)

&(按位与)是把两个数的二进制按位与，两个为真才为真(1&1=1)

&&(逻辑与)

先运算左边的表达式，一旦为假，后续不管有多少表达式，均不在计算，结果为假，
一个为真才计算右边的表达式，全为真才为真

|(按位或运算)

|(按位或)也是把两个数的二进制数按位做或运算, 一个为真即为真(1|0=1)

||(逻辑或)

一个为真即为真，后续不在计算，为假在计算后边的表达式

^(异或运算)

异或运算，相同为0，不同为1，1^0=1, 1^1=0

<<(左移运算符)

示例：5<<2
把5的二进制位左移两位，右边补0, 5的二进制数为0000 0101，把有效值101左移
两位就是0001 0100， 左边第一位正数补0，负数补1，等于乘了2多少次方

>>(右移运算符)

示例：5>>2
同样是对二进制数右移2位，0000 0101, 右移两位就是0000 0001，左边第一位正数补0，
负数补1，等于除以2的n次方

~(取反运算符)

把二进制数按位取反

>>>(无符号右移运算符)

>>>(无符号右移运算符)与>>(右移运算符)的主要区别在于负数的运算，正数无符号右移是高位补0，
移多少位补多少0
对于负数的无符号右移：
示例：-6>>>3
6的二进制取反在加1:  
  0000 0000 0000 0000 0000 0000 0000 0110 取反
  1111 1111 1111 1111 1111 1111 1111 1001 加1
  1111 1111 1111 1111 1111 1111 1111 1010
右移三位 
  0001 1111 1111 1111 1111 1111 1111 1111
